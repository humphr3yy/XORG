# XORG - Pygame Edition

A Python/Pygame port of the XORG arena dueling game.

## Requirements
- Python 3.x
- Pygame (`pip install pygame`)

## Setup & Running
First, run the setup script to create a virtual environment and install dependencies:
```bash
./setup_env.sh
```

Then run the game:
```bash
source venv/bin/activate
python3 main.py
```

## Controls
- **Mouse**: Aim
- **Left Click**: Shoot
- **Movement**: Physics-based recoil from shooting
